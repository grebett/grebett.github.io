GBT-0104

Index bleu

Un index manuscrit classe plusieurs entrées avec une couleur qui ne correspond à aucune catégorie connue.

14/06/2026 note text
noteindexcouleur

La couleur bleue ne signale pas l’eau, ni la nuit, ni les sons.

Elle semble plutôt indiquer les documents qui ont changé de place après leur classement.
